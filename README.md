# Meu Álbum da Copa 2026 — v1.0

Projeto limpo/mobile-first para controle de figurinhas da Copa 2026.

## O que esta versão mantém

- Base `data.js` com seleções, ordem do álbum, nomes e tipos das figurinhas.
- Total de 994 figurinhas.
- Controle de quantidade, faltantes, repetidas, trocas e backup.
- PWA com manifest e service worker.
- Sincronização Google/Firebase quando `firebase-config.js` estiver configurado.

## Deploy na Vercel

1. Crie um repositório novo no GitHub.
2. Suba os arquivos extraídos deste ZIP, não o ZIP fechado.
3. Na Vercel, importe o repositório.
4. Use:
   - Build command: `npm run build`
   - Output directory: `dist`
5. Publique.
