## Troca manual
Para quem ainda não usa o app. A pessoa manda as listas por WhatsApp e o usuário cola no app.

## Código Trocaí
Preparado visualmente para a próxima evolução. Cada álbum terá um código público/semipúblico de troca para comparação automática.
