# Blindagem de uso real — v1.5.6

## O que foi protegido

### Backup JSON
- Exportar backup agora usa um pacote com metadados, resumo e estado.
- Importar/restaurar JSON exige confirmação detalhada.
- Antes de importar, o app oferece exportar backup de segurança.

### Álbum familiar
- Ao criar um familiar, o app pergunta se deve começar com o álbum atual ou zerado.
- Foi adicionado “Copiar pessoal → familiar”.
- Entrar em um familiar oferece backup antes de trocar o modo.
- Textos deixam claro que JSON é backup e família é sincronização compartilhada.

### Sincronização
- Perfil mostra modo atual e última alteração local.
- Confirmações ficaram mais explícitas para ações destrutivas.

## Regras de uso
- Exportar backup antes de importar JSON, zerar coleção ou copiar pessoal para familiar.
- Usar JSON apenas como restauração manual.
- Usar álbum familiar para sincronização entre contas.
